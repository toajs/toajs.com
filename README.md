ToaJS.com
====
The toajs.com website.


本项目主要作为 ToaJS 官网，也是 Toa 示例项目，主要提供如下功能：

1. 作为开发环境，提供 view 和静态资源驱动服务；
2. 为前端内容提供打包服务；

你只需要复制本项目，然后根据需求修改成自己的项目即可。

## usage

**development:**

```bash
npm install
bower install
gulp build
pm2 start pm2/release.json
```

访问：[http://localhost:3000](http://localhost:3000)
